Top Url REST API

Swagger docs: https://app.swaggerhub.com/apis/lemk1/top_url_api/1.0.0#/

Detailed documentation

Top URL REST API has useful and detailed yaml Swagger documentation for each endpoint and models. Docs shows REST API usage step by step. You can read the docs here: https://app.swaggerhub.com/apis/lemk1/top_url_api/1.0.0#/

Open-source code at GitHub

Top Url service is free open-source project wrote on Python3.7 and used Sanic web-framework, PostgreSQL, lemkpg, and others modern technologies. You can view each line in the code, and clone or fork repository if you want. You can view source code here: https://github.com/Daniil-Kost/top_url

Simple usage

You can usage Top Url REST API with any comfortable tools for. You can work with this API through Postmen, or with another tool for json requests
